# Before Start
- Create Firestore
- Create .env file copy my '.env.example' and set your config
- Create collection (in my case i created as 'blogs')

# Start Setup

```
$ npm install
or
$ yarn install
```

# Tech Stack
- React
- Tailwind
- React Router
